*JIKA INGIN MENGGUNAKAN APLIKASI INI DI LAPTOP/PC LAIN, BACA ATURAN DI BAWAH INI:

1. Instal semua jenis fonts, yang berada di dalam folder aplikasi dengan nama folder 'Fonts'.
2. Masukkan database yang bernama "db_ukk_mobil.bak " yang ada di dalam folder aplikasi dengan nama folder 'Database' ke SQL SERVER 2012.
3. Setelah langkah 1 & 2 sudah selesai, maka sekarang langsung buka aplikasi menggunakan VISUAL BASIC 2012, jangan langsung dijalankan, ubah terlebih dahulu nama lokasi database di setiap form             loadnya. CONTOH:
    klik modules-mobil-klik double pada form-lalu lihat kode seperti ini:  cn.ConnectionString = "Data Source=RISTI\SQLEXPRESS;Initial Catalog=db_ukk_mobil;User ID=sa;Password=sa2008"
    yang berada dalam tanda petik disesuaikan dengan nama sql server, nama database, nama user, dan password yang digunakan.
    PERHATIKAN DI SETIAP FORM LOAD DI DALAM APLIKASI!
4. Jika langkah 1-3 sudah selesai dan tidak ada masalah, maka aplikasi "SISTEM PEMBELIAN MOBIL" bisa langsung dijalankan dengan menekan tombol "Start/F5".
